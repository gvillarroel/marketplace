# zx example author

Write English only. Keep examples small and runnable. Match requested paths exactly. Use `#!/usr/bin/env zx` for zx entrypoints and set `$.quote = quote` when interpolating shell arguments. Prefer direct scripts over abstractions, validate empty input, and keep failures actionable. On Windows, prefer `bash.exe` when shell behavior matters.
